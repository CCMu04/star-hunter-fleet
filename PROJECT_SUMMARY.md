# 星际猎人配队站 - 项目总结

## 项目概述
这是一个为《星际猎人》（原《无尽的拉格朗日》）游戏开发的一站式配队平台，整合完整的舰船数据、智能配队推荐、蓝图加点模拟等功能。

## 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI 组件库**: Ant Design 5
- **样式方案**: Tailwind CSS
- **状态管理**: Zustand
- **数据可视化**: ECharts

## 数据详情

### 舰船数据 (ships.json)
包含以下字段：
- 基础信息：名称、稀有度、势力、类型、武器类型、定位、攻击属性、站位、获取类型
- 战斗属性：对舰火力、对空火力、攻城火力、结构值、抵抗伤害、护盾值、巡航速度、曲率速度、仓储容量
- 性能评级：对舰、防空、攻城、支援、生存、战略（S/A/B/C 级）
- 建造信息：金属成本、晶体成本、重氢成本、建造时间、维修速度
- 其他：指挥值、服役上限、旗舰技能、标签、模块系统、Wiki链接
- 额外信息：职业分类、长度、实装日期

### 势力数据 (factions.json)
包含 9 个势力，每个势力有独立的配色方案。

## Excel 数据处理

如果需要重新处理 Excel 数据：

```bash
node --no-warnings --experimental-modules scripts/parse-excel.js
```

这个脚本会：
1. 读取 Excel 文件
2. 解析所有 199 艘舰船数据
3. 保存为 ships.json 和 factions.json
4. 同时保存原始数据到 raw-ship-data.json
